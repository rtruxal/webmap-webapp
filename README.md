Webmap
======

## Descr!

When it's complete, this repo will contain an interactive webapp that maps websites which use the same IP address.  



## Requirements:
 - python 3.7, with:
   - Flask
   - graphene
   - py2neo
   - flask_graphql
 - neo4j installed & running on bolt://localhost:\<someport>
 - a browser
